# Flutter Module

Gestión de proyectos [Flutter](https://flutter.dev).

## Tasks

### `flutter:check`
Verifica Flutter y Dart.

### `flutter:setup`
Instala dependencias.

```bash
flutter pub get
```

### `flutter:fmt`
Formatea código Dart.

```bash
dart format .
```

### `flutter:fix`
Aplica fixes recomendados.

```bash
dart fix --apply
```

### `flutter:lint`
Analiza código Dart.

```bash
dart analyze
```

### `flutter:test`
Ejecuta tests.

```bash
flutter test
```

### `flutter:build`
Build APK release.

```bash
flutter build apk --release
```

## Dependencias

- `flutter`
- `dart`
