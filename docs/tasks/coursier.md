# Coursier Module

Gestión de dependencias Scala con [Coursier](https://get-coursier.io).

## Tasks

### `coursier:check`
Verifica Coursier.

### `coursier:setup`
Instala Coursier CLI.

```bash
sudo wget -nv -O /usr/local/bin/cs https://git.io/coursier-cli
sudo chmod +x /usr/local/bin/cs
```

## Dependencias

- `cs` (Coursier CLI)
